# 관리자 대시보드 작업 맥락 노트

## 2026-05-18

- 관리자 기능은 한 번에 한 페이지씩 진행한다.
- 1차 완료 기준은 백엔드 API, 관리자 로그인, `/admin` 최소 화면 연결까지다.
- 관리자 코드는 MVP 시연용으로 `admin2026` 고정값을 사용한다.
- 대시보드 데이터는 실제 DB만 사용하고, 샘플 fallback은 두지 않는다.
- 대시보드 레이아웃은 답변 대기 문의를 크게 배치하는 문의 우선형으로 결정했다.
- `/admin/agendas`, `/admin/inquiries`는 이번 단계에서 페이지를 만들지 않고 링크만 남긴다.
- 기존 미추적 `AGENTS.md`는 사용자 작업으로 보고 건드리지 않는다.
