# 관리자 기능명세서

## 4.1 관리자 대시보드

### 4.1.1 관리자 전용 GNB
- 어두운 배경 네비게이션 바
- 로고 + '관리자' 배지 + 대시보드·안건 관리·문의 관리 탭
- 현재 탭 파란 배경 강조
- **수용 기준**
  - 관리자 계정으로만 접근 가능
  - 일반 사용자 GNB와 구분되는 다크 테마
  - 현재 탭 활성 스타일
- **엣지 케이스**: 일반 사용자가 관리자 URL 직접 접근 시 차단 + 리다이렉트
- **비고**: 관리자 전용 다크 테마
- **사용자**: 관리자
- **중요도**: 높음

### 4.1.2 핵심 지표 카드
- 가로 4등분 카드:
  - 전체 안건 (이번 주 신규 수 표시)
  - 답변 대기 문의 (미처리 수 강조)
  - 검토 중 안건
  - 반영 완료 (누적)
- **수용 기준**: 수치 실시간 또는 주기적 업데이트, '답변 대기' 카드는 긴급성 강조 색상(주황) 권장
- **엣지 케이스**: 답변 대기 없으면 초록 색상으로 변경 고려
- **사용자**: 관리자
- **중요도**: 높음

### 4.1.3 공감도 높은 안건 섹션
- 찬성률 기준 상위 안건 목록
- 각 항목: 안건명 / 공감률 바 / 공감률% / 처리 상태 배지
- '전체 보기 →' 링크
- **수용 기준**
  - 찬성률 내림차순 정렬
  - '전체 보기 →' 클릭: 안건 관리 화면으로 이동
- **엣지 케이스**: 안건 없음 시 '등록된 안건이 없습니다' 빈 상태
- **사용자**: 관리자
- **중요도**: 높음

### 4.1.4 답변 필요한 문의 섹션
- 대기 중 문의 목록
- 각 항목: 문의명 / 유형·익명 / 날짜 / 상태 배지
- '전체 보기 →' 링크
- **수용 기준**
  - 등록일 기준 오래된 것 우선 표시
  - '전체 보기 →' 클릭: 문의 관리 화면(대기 필터) 이동
- **엣지 케이스**: 대기 문의 없음 시 '답변 대기 문의가 없습니다' 표시
- **사용자**: 관리자
- **중요도**: 높음

### 4.1.5 통계 바 차트
- 카테고리별 안건 수 / 처리 상태별 분포 / 문의 유형별 현황을 가로 바 차트로 표시
- **수용 기준**: 바 길이는 최대값 기준 상대적 비율로 표시, 값 우측 숫자 표시
- **엣지 케이스**: 데이터 없는 항목은 바 없이 0 표시
- **사용자**: 관리자
- **중요도**: 중간

## 4.2 안건 관리

### 4.2.1 안건 검색 박스
- 키워드로 안건 제목 검색, 실시간 또는 엔터로 실행
- **수용 기준**
  - 검색어로 제목 필터링
  - 검색어 지우면 전체 목록 복귀
- **엣지 케이스**: 검색 결과 없음 시 '검색 결과가 없습니다' 빈 상태
- **사용자**: 관리자
- **중요도**: 중간

### 4.2.2 처리 상태 필터 칩
- 전체 / 접수됨 / 검토 중 / 전달 완료 / 반영 완료 / 반영 어려움 칩 버튼
- **수용 기준**
  - 기본값: '전체'
  - 클릭 시 해당 상태 안건만 표시
  - 검색과 AND 조건으로 적용
- **사용자**: 관리자
- **중요도**: 높음

### 4.2.3 안건 목록 테이블
- 컬럼: 안건 제목·등록일 / 찬성 수 / 반대 수 / 공감률 / 카테고리 배지·학과명 / 처리 상태 변경 드롭다운
- **수용 기준**
  - 공감률 내림차순 기본 정렬 (정책 결정)
  - 공감률 파란색 굵은 폰트 강조
- **엣지 케이스**: 안건 없음 시 '등록된 안건이 없습니다' 빈 상태
- **사용자**: 관리자
- **중요도**: 높음

### 4.2.4 처리 상태 변경 드롭다운
- 각 안건 행의 인라인 드롭다운
- 옵션: 접수됨 / 검토 중 / 학교 전달 완료 / 반영 완료 / 반영 어려움
- 선택 즉시 저장
- **수용 기준**
  - 변경 즉시 저장 (별도 저장 버튼 없음)
  - 변경 즉시 학생 화면 처리 타임라인에 반영
  - 변경 이력 저장 권장
- **엣지 케이스**: 변경 실패 시 이전 상태로 롤백 + 오류 토스트
- **비고**: 이력 저장 권장
- **사용자**: 관리자
- **중요도**: 높음

## 4.3 문의 관리

### 4.3.1 문의 목록 패널 (좌측)
- 전체 / 답변 대기 / 검토 중 / 답변 완료 탭 필터
- 각 항목: 상태 배지 / 유형 / 날짜 / 제목 / 내용 미리보기
- 선택된 문의 파란 왼쪽 테두리 강조
- **수용 기준**
  - 기본값: '전체' 탭
  - 좌우 분할 레이아웃
  - 답변 대기 우선 정렬 권장
- **엣지 케이스**: 문의 없음 시 빈 상태 표시
- **비고**: 좌우 분할 레이아웃
- **사용자**: 관리자
- **중요도**: 높음

### 4.3.2 문의 상세 헤더
- 우측 패널: 제목 / 유형·익명·등록일 / 상태 배지 표시
- **수용 기준**: 선택한 문의의 전체 정보 표시
- **엣지 케이스**: 작성자 탈퇴 시 '탈퇴한 사용자' 표시 정책 결정
- **사용자**: 관리자
- **중요도**: 높음

### 4.3.3 문의 내용 표시
- 회색 박스 안에 문의 내용 전체 텍스트, 읽기 전용
- **수용 기준**: 줄임 없이 전체 내용 표시, 스크롤 가능한 박스
- **사용자**: 관리자
- **중요도**: 높음

### 4.3.4 상태 변경 드롭다운
- 답변 작성 전 상태 선택
- 옵션: 검토 중 / 답변 완료
- **수용 기준**
  - 답변 등록 시 선택된 상태로 자동 변경
  - 변경 즉시 학생 내 문의함에 반영
- **엣지 케이스**: 답변 없이 상태만 변경 허용 여부 정책 결정
- **사용자**: 관리자
- **중요도**: 높음

### 4.3.5 답변 텍스트에어리어
- 학생에게 전달할 답변 입력
- '답변 내용은 해당 학생에게만 비공개로 전달됩니다' 안내 포함
- 최대 글자 수 TBD
- **수용 기준**: 비어있으면 '답변 등록하기' 버튼 비활성
- **사용자**: 관리자
- **중요도**: 높음

### 4.3.6 답변 등록하기 버튼
- 답변 내용 입력 시 활성화
- 클릭 시 답변 저장 + 문의 상태 변경 + 학생 내 문의함 반영
- 하단 안내: '등록 후에도 수정할 수 있습니다.'
- **수용 기준**
  - 내용 없으면 비활성
  - 등록 성공: 성공 토스트 + 목록 상태 업데이트
  - 등록 실패: 오류 토스트 + 내용 유지
- **엣지 케이스**: 중복 제출 방지(비활성화 후 처리), 수정 시 기존 답변 덮어쓰기 또는 버전 관리 정책 결정
- **사용자**: 관리자
- **중요도**: 높음
