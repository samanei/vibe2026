# 캠퍼스 보이스 기획서

## 1. 프로젝트 개요

- **서비스명**: 천마광장
- **한 줄 소개**: 학생들의 흩어진 의견을 공개 안건과 비공개 문의로 정리하는 익명 기반 학내 소통 플랫폼

천마광장은 대학생들이 학교생활 중 느끼는 불편사항과 개선 의견을 익명으로 제안하고, 다른 학생들의 투표를 통해 공감도를 확인할 수 있는 서비스다.

공개적으로 논의할 수 있는 문제는 **공개 안건**으로 등록해 찬성/반대 투표를 받고, 개인적이거나 민감한 질문은 **비공개 익명 문의함**을 통해 학생회나 담당자에게 전달한다.

학생회는 관리자 화면에서 공감도 높은 안건과 답변 대기 문의를 확인하고, 처리 상태를 관리할 수 있다.

---

## 2. 문제 정의

현재 대학 내 학생 의견은 에브리타임, 학과 단톡방, 총학생회 카카오톡 채널 등 여러 곳에 흩어져 있다.

하지만 이런 방식은 의견을 자유롭게 말할 수는 있어도, 실제 안건으로 정리하거나 처리 상태를 확인하기 어렵다.

**핵심 문제**: 학생들의 의견은 존재하지만, 이를 부담 없이 제안하고, 다수 의견인지 검증하며, 학생회가 체계적으로 관리할 수 있는 구조가 부족하다.

---

## 3. 타겟 사용자

### 일반 학생
- 학교생활 중 불편함이나 개선 의견이 있는 학생
- 실명으로 학생회나 학교에 직접 문의하기 부담스러운 학생
- 다른 학생들도 같은 문제를 느끼는지 확인하고 싶은 학생
- 공개하기 어려운 개인 문의를 익명으로 전달하고 싶은 학생

### 총학생회 / 학과 학생회 / 학교 담당자
- 학생 의견을 체계적으로 수집하고 싶은 운영 주체
- 어떤 안건이 다수 학생의 공감을 받는지 확인하고 싶은 학생회
- 반복 문의와 주요 불편사항을 한 곳에서 관리하고 싶은 담당자

---

## 4. 솔루션

천마광장은 학생 의견 전달 방식을 **공개 안건 제안**과 **비공개 익명 문의**로 분리한다.

공개 안건은 학생들이 익명으로 제안하고, 다른 학생들이 찬성/반대 투표를 통해 공감도를 표현한다.

비공개 익명 문의는 공개 게시판에 올리기 어려운 개인적 질문을 학생회나 담당자에게 전달하는 기능이다.

학생회는 관리자 화면에서 공개 안건과 비공개 문의를 구분해 확인하고, 공감도 높은 안건의 처리 상태를 변경하거나 비공개 문의에 답변할 수 있다.

## 5. 주요 기능

### 학생용 기능

| 기능 | 설명 |
|------|------|
| 학교/학과 기반 가입 | 이름·학번 없이 학교와 학과만 선택 |
| 공개 안건 작성 | 시설, 수업, 행정, 복지 등 학교생활 개선 의견 작성 |
| 안건 목록/상세 조회 | 다른 학생들이 올린 안건과 공감도 확인 |
| 찬성/반대 투표 | 안건에 대한 학생들의 공감 여부 확인 |
| 처리 상태 확인 | 접수됨, 검토 중, 학교 전달 완료, 반영 완료 등 확인 |
| 비공개 문의 작성 | 개인적이거나 민감한 문의를 익명으로 전달 |
| 내 문의함 | 문의 답변 상태와 답변 내용 확인 |

### 관리자/학생회 기능

| 기능 | 설명 |
|------|------|
| 관리자 대시보드 | 전체 안건, 답변 대기 문의, 인기 안건 확인 |
| 공개 안건 관리 | 공감도 높은 안건 확인 및 우선순위 판단 |
| 안건 상태 변경 | 접수됨, 검토 중, 학교 전달 완료, 반영 완료 등 변경 |
| 비공개 문의 관리 | 학생 문의 확인 및 답변 작성 |

---

## 7. 차별점

1. **단순 익명 게시판이 아닌 안건화 구조** — 투표를 통해 공감도를 확인
2. **공개 안건과 비공개 문의 분리** — 공론화와 개인 문의를 구분
3. **처리 상태 기반 투명성 제공** — 접수 → 검토 → 전달 → 반영 추적 가능
4. **학생회 관리 효율화** — 찬성 수와 공감률 기준 우선순위 판단

---

## 8. 기대 효과

### 학생 측면
- 실명 문의 부담 감소
- 학교생활 불편사항을 쉽게 제안 가능
- 다른 학생들의 공감 여부 확인 가능
- 개인적 문의를 비공개로 전달 가능
- 안건 처리 상태 확인 가능

### 학생회/학교 측면
- 흩어진 학생 의견을 한 곳에서 관리 가능
- 공감도 높은 안건을 빠르게 파악 가능
- 답변 대기 문의를 체계적으로 관리 가능
- 학교 측에 전달할 근거 데이터 확보 가능
- 학생회 운영 업무 효율화

---

## 9. 최종 요약

천마광장은 대학 내 학생들의 흩어진 의견을 **공개 안건**과 **비공개 문의**로 나누어 수집하고 관리하는 익명 기반 학내 소통 플랫폼이다.

이번 해커톤 MVP에서는 실시간 채팅과 고도화된 인증 기능은 제외하고, **공개 안건 등록·투표·상태 관리·비공개 문의 답변**이라는 핵심 흐름을 구현한다.
