# 비공개 문의 기능명세서

## 3.1 문의 작성

### 3.1.1 보안 안내 배너
- 초록 배경 배너: '🔒 학생회/관리자만 확인할 수 있습니다. 다른 학생에게는 절대 공개되지 않습니다.'
- **수용 기준**: 정적 안내 배너, 항상 표시
- **중요도**: 높음

### 3.1.2 문의 유형 드롭다운
- 선택 옵션: 장학금 / 학비 / 휴학 / 복학 / 행사 / 부스 신청 / 학생회 운영 / 제보 / 기타
- 필수(*)
- **수용 기준**: 필수 항목, 미선택 시 제출 불가
- **엣지 케이스**: 옵션 변경 시 기존 데이터 마이그레이션 정책 결정
- **중요도**: 높음

### 3.1.3 제목 입력 필드
- 텍스트 입력, 필수(*), 최대 글자 수 TBD
- **수용 기준**: 필수 항목, 미입력 시 제출 불가
- **엣지 케이스**: 공백만 입력 시 제출 시 유효성 검증으로 차단
- **중요도**: 높음

### 3.1.4 문의 내용 텍스트에어리어
- 상세 내용 입력, 필수(*), 최대 1,000자
- **수용 기준**
  - 필수 항목, 미입력 시 제출 불가
  - 최대 1,000자 초과 차단
  - 글자 수 카운터 표시
- **엣지 케이스**: 최대 글자 수 초과 붙여넣기 시 제한까지만 허용
- **중요도**: 높음

### 3.1.5 안내 사항 박스
- 4가지 안내 문구:
  - 📬 제출 후 '내 문의함'에서 답변 확인 가능
  - ⏱ 평균 답변 기간 3~5 영업일 이내
  - 🔒 담당자 외 열람 불가
  - ⚠ 허위·악의적 문의는 처리 거부 가능
- **수용 기준**: 정적 안내 박스, 항상 표시. '내 문의함' 텍스트 링크 처리 가능
- **중요도**: 중간

### 3.1.6 문의 제출하기 버튼
- 필수 항목(유형·제목·내용) 모두 입력 시 활성
- **수용 기준**
  - 모든 필수 항목 입력 시 활성
  - 제출 성공: 내 문의함 이동 + 성공 토스트
  - 제출 실패: 오류 메시지 + 재시도
- **엣지 케이스**: 중복 제출 방지(버튼 비활성화 후 처리), 네트워크 오류 시 오류 토스트 + 입력값 유지
- **중요도**: 높음

## 3.2 내 문의함

### 3.2.1 상단 버튼
- 좌측 '← 돌아가기' 버튼: 공개 안건 목록으로 이동
- 우측 '+ 새 문의' 버튼: 문의 작성 화면으로 이동
- **수용 기준**: 각 버튼 클릭 시 해당 화면으로 이동
- **중요도**: 높음

### 3.2.2 상태 필터
- 전체 / 답변 대기 / 검토 중 / 답변 완료 항목
- **수용 기준**
  - 기본값: '전체' 선택
  - 클릭 시 해당 상태만 필터링
- **엣지 케이스**: 해당 상태 문의 없음 시 빈 상태 메시지 표시
- **중요도**: 중간

### 3.2.3 내 현황 위젯
- 총 문의 / 답변 완료 / 대기 중 / 검토 중 카운트 카드 표시
- **수용 기준**: 본인 문의 통계만 표시, 실시간 카운트 업데이트
- **중요도**: 중간

### 3.2.4 문의 카드 — 답변 완료
- 상태 배지(초록) / 유형 / 날짜 / 제목 / 내용 미리보기
- 답변 박스: 작성자·날짜·답변 전체, 파란 왼쪽 테두리로 구분
- **수용 기준**
  - 답변 완료 상태에서만 답변 박스 표시
  - 답변 내용 전체 표시(줄임 없음)
- **엣지 케이스**: 답변이 삭제된 경우 '답변이 삭제되었습니다' 표시 정책 결정
- **중요도**: 높음

### 3.2.5 문의 카드 — 검토 중 / 답변 대기
- 각 상태 배지(파랑/주황) / 유형·날짜·제목·내용 미리보기 표시
- 답변 박스 없음
- **수용 기준**
  - 답변 없는 상태에서는 답변 박스 미표시
  - 내용 미리보기 말줄임 처리
- **엣지 케이스**: 검토 중 → 답변 완료 전환 시 답변 박스 즉시 표시
- **중요도**: 높음
